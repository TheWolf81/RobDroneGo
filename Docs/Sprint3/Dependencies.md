# Estratégia de organização do trabalho
* US's de alta prioridade, uma vez que têm outras dependentes delas, terão que ser discutidas com todos os membros do grupo para que haja uniformidade na implementação. Após análise e decisões, serão divididas pelos membros do grupo.


* US's de média prioridade serão divididas pelos membros do grupo, a duas pessoas por US ou grupo de US's de forma a que haja um "buddy system", com o objetivo de validação mútua do trabalho.


* US's de baixa prioridade são divididas por todos os elementos do grupo, para que todos tenham algo para fazer nos momentos de menor capacidade de trabalho em grupo
